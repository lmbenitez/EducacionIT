FLEXBOX

Es una caracteristica que nos ofrece css3, la cual permite la alineacion de etiquetas utilizando como base al contenedor, sin necesidad de modificar la visualizacion del contenido.

Justify-Content: Permite acomodar las etiquetas contenidas de manera horizontal

  • Flex-Start: Alinea las etiquetas al comienzo del flex
  • Flex-End: Alinea las etiquetas al final del flex
  • Center: Centra las etiquetas horizontalmente
  • Space-around: Distribuye las etiquetas de manera espaciada, procurando que todas tengan los mismo margenes (los margenes de separacion entre etiquetas se suman)
  • Space-between: Distribuye las etiquetas de manera espaciada alineando la primer y ultima etiqueta hacia los extremos(los margenes internos son iguales para todas las etiquetas, pero no hay margenes externos para las etiquetas de los extremos)
  • Space-evenly: Distribuye las etiquetas de manera espaciada, calculando exactamente la misma distancia de margen para cada una (Caracteristica NO compatible con Navegadores como Internet Explorer)

Align-Items: Permite acomodar las etiquetas contenidas de manera vertical

  • Flex-Start: Alinea las etiquetas al comienzo del flex
  • Flex-End: Alinea las etiquetas al final del flex
  • Center: Centra las etiquetas verticalmente
  • Stretch: Fuerza a todas las etiquetas contenidas a que ocupen todo el alto de la etiqueta contenedora
  • Baseline: Centra las etiquetas en base al alto de la linea
