Responsive Web Design

Este sitio fue desarrollado con fines educativos, para el aprendizaje y entendimiento de las diferentes practicas relacionadas al Desarrollo UI/UX.

  • Fluid Web Design
  • Adaptive Web Design
  • Responsive Web Desgin

Se utilizaron las herramientas provistas por los metodos previamente mensionados e implementando el framework desarrollado por Twitter, ideal para las practicas responsive, tambien conocido como Bootstrap.

Autor: Cristian Racedo
